# Avito Tech Internship Project

## Описание проекта

Веб-приложение для просмотра и редактирования объявлений с расширенным функционалом:

* фильтрация и поиск
* редактирование карточки товара
* генерация описания и рекомендации цены с помощью AI
* переключение отображения списка
* поддержка светлой и тёмной темы

---

# Стек технологий

## Frontend

* React
* TypeScript
* React Hook Form
* Mantine (UI + hooks)
* MobX (стейт-менеджмент)
* TanStack Query (работа с API)

## Backend

* Node.js
* Fastify
* Zod (валидация)
* Axios (HTTP-запросы)

## AI

* GigaChat API (Sber)

---

# 🛠 Установка и запуск

## 1. Клонирование проекта

```bash
git clone <repo-url>
cd avito-tech-internship
```

---

## 2. Установка зависимостей

### frontend

```bash
cd client
npm install
```

### backend

```bash
cd ../server
npm install
```

---

## 3. Настройка переменных окружения (LLM)

Создай файл:

```
server/.env
```

Добавь:

```env
GIGACHAT_AUTHORIZATION_KEY=your_key_here
```

---

## 5. Запуск проекта

### backend

```bash
cd server
port=8080 npm run start
```

### frontend

```bash
cd client
npm run dev
```

---

# AI (GigaChat)

Backend:

* автоматически получает OAuth токен
* кэширует его до истечения срока
* обновляет при необходимости

Используется для:

* генерации описания товара
* оценки рыночной цены

---

# ✨ Дополнительный функционал

## Светлая и тёмная тема

* реализовано через Mantine
* переключение через кнопку (Affix)
* сохраняется в `localStorage`
* автоматически применяется при перезагрузке

---

## Переключение вида отображения

* grid / list режим
* управление через `SegmentedControl`
* хранится в MobX store
* сохраняется в `localStorage`

---

## Рекомендация цены от AI

* на основе параметров товара
* выводит диапазоны:

  * отличное состояние
  * среднее состояние
  * дефекты / срочная продажа
* можно применить значение в форму

---

## Генерация описания

* AI генерирует текст объявления
* учитывает:

  * название
  * категорию
  * характеристики
* можно:

  * принять
  * отклонить
  * перегенерировать

---

## Автосохранение формы

* данные формы сохраняются в `localStorage`
* при перезагрузке не теряются

---

# Особенности реализации

* динамическая форма параметров по категории
* валидация без сторонних схем (через react-hook-form)
* обработка загрузки и ошибок API
* дебаунс поиска
